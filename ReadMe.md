## 🌐 Portfolio Features

The portfolio itself is a hand-crafted single-file web app with several unique features:

### 🕸️ Interactive Knowledge Graph
- **160+ nodes, 350+ edges** mapping every skill, project, course, tech, certification, and concept
- Filter clusters: AI/NLP · ML · iOS · Web · Game Dev · Systems · Competitive · Courses · Math · Certs
- Hover to explore connections · Click to pin a node · Double-click two nodes to find path
- Auto-explore mode cycles through the most-connected nodes automatically
- Live search across node labels and descriptions

### 📚 Academic Log (6 Semesters)
- Full semester-by-semester breakdown of every course at IIIT-Delhi
- Each course includes: description, key topics, personal learnings/insights, and real-world applications
- SGPA/CGPA progression charts with animated SVG line graphs

### 🖥️ Terminal Aesthetic
- CRT scanline overlay, JetBrains Mono typography, custom dark theme
- Custom animated cursor with magnetic ring
- Typewriter intro animation, glitch effect on name
- Noise texture, nav progress bar, scroll-reveal animations

### 💡 Recruiter-Optimised Layout
- "Open to Work" status, notice period, and preferred roles displayed prominently
- Featured Work strip for instant 3-second recruiter scan
- Contact availability JSON terminal block
- Quick-access links to experience letters and certificates

### 🎨 Animations & Interactions
- Magnetic card tilt on hover (3D perspective)
- Skill bar fill animations on scroll into view
- Staggered grid entrance on page transitions
- Counter animations for stats (CGPA, problems, rating)
- KG filter cascade-reveals clusters with flash wave
